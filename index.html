01
Data Python Pandas

E-Commerce Sales Trend Analysis

Problem

A retail dataset with 50,000+ transaction records had no clear insight into which products, regions, or time windows drove revenue — or why certain months showed sudden drops.

Approach

Cleaned the raw CSV using Pandas (handled nulls, date parsing, outlier removal). Built a 5-step analysis pipeline: cohort analysis → product-level aggregation → seasonal decomposition → correlation matrix → visual storytelling.

💡 Key Insights
  • Top 8% of products drove 63% of total revenue — classic Pareto pattern confirmed
  • Revenue dips in Feb/March traced to inventory delays, not demand drops
  • Repeat purchase rate was 2.4× higher for customers acquired via referral
Pandas · NumPy · Seaborn · Matplotlib

Student Assistant App

Problem

03
IoT Arduino C++

Smart Gas & Fire Alert System

Problem

Domestic gas leaks and fire hazards often go undetected until too late. Existing commercial solutions are expensive and not customizable for small households.

Approach

Built a dual-sensor IoT system using MQ-2 (gas sensor) + flame sensor with Arduino Uno. Alerts fire via buzzer + LED + optional SMS (GSM module). Threshold calibrated through multiple burn tests to minimize false positives.

💡 Key Insights
  • False positive rate dropped from 22% to <3% after threshold tuning with logged test data
  • Response latency under 2 seconds from detection to alert — critical for safety
  • Total build cost: ₹380 vs ₹2,500+ for commercial equivalents
Arduino · C++ · MQ-2 Sensor · GSM Module

04
Data Python Prediction

Student Performance Prediction Model

Problem

Identifying at-risk students early enough to intervene is hard — teachers rely on intuition. Is there a data-driven way to predict academic performance before final exams?

Approach

Used UCI Student Performance dataset (395 records, 30 features). Feature engineering on attendance, parental education, study time. Built a Linear Regression baseline and improved with Random Forest. Evaluated with MAE and R² score.

💡 Key Insights
  • Study time and past failures were the two strongest predictors — not family income
  • Random Forest reduced MAE by 31% over baseline linear model
  • Alcohol consumption had a statistically significant negative impact (correlation: -0.37)
Python · Scikit-learn · Pandas · Seaborn

05
Data EDA Excel

IPL Match Data Exploratory Analysis

Problem

IPL data is rich but messy — multiple seasons, player trades, venue changes. What actually determines whether a team wins a close match?

Approach

Cleaned 11 seasons of Kaggle IPL data. Performed EDA across team win rates, toss decisions, venue impact, and powerplay run rates. Built a correlation analysis between toss results and final outcomes.

💡 Key Insights
  • Toss winners choose to field first 68% of the time — and win those matches at 54%
  • Chasing is slightly more successful at neutral venues vs home grounds
  • Powerplay economy rate (overs 1–6) is a stronger win predictor than death overs
Python · Pandas · Matplotlib · Excel
